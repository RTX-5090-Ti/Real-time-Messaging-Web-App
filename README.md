# Real-time Chat App (React + Express + Socket.IO)

A real-time 1–1 chat web app with JWT-cookie authentication, friends system, notifications, file upload (Cloudinary), GIF sending (GIPHY), typing/seen indicators, and online/offline presence.

## Tech Stack

- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express (MVC)
- Database: MongoDB (Mongoose)
- Realtime: Socket.IO
- Auth: JWT Cookie
- Upload: Cloudinary
- GIF: GIPHY API

## Features

- Register / Login / Logout (JWT Cookie)
- Real-time messaging (Socket.IO)
- Typing indicator, Seen/Read indicator
- Online/Offline presence
- Direct 1–1 conversations
- Message history with pagination (load more)
- Search messages in a conversation
- Friends system (request / accept / reject) + search user by email
- Notifications (realtime + mark all as read)
- File/Image upload (Cloudinary) + attachments preview
- Send GIF (GIPHY)
